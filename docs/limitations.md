# Limitations

## Synthetic Case Design

The current benchmark uses researcher-designed synthetic cases. This provides control, but it also creates the risk that evaluation reflects researcher expectations.

## Researcher-Coded Scores

The current scores were assigned by the research workflow. Independent blinded human evaluation is needed before making strong empirical claims.

## MCDA Calibration

MCDA weights and thresholds are provisional. A manuscript-ready version should justify weights using literature, expert elicitation, or sensitivity analysis.

## Compact Run Limits

Compact runs are useful for broad cross-case comparison, but they are weaker than full runs for proving deliberative process quality. At least one additional full run is recommended.

## No Real-World Validation Yet

The benchmark does not yet include real reservoir cases. Real cases are necessary for external validity.
