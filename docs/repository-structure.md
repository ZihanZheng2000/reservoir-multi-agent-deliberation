# Repository Structure

| Directory | Purpose |
|---|---|
| `cases/` | Scenario packets and evaluator-only gold checklists. |
| `runs/` | Model outputs organized by case and run type. |
| `protocols/` | Evaluation protocols and benchmark design documents. |
| `reports/` | Cross-case reports and manuscript-style summaries. |
| `docs/` | Human-facing project documentation. |
| `packages/` | Planning and modeling packages from the research workflow. |
| `artifacts/` | Original working artifacts retained for traceability. |
| `archive/` | Reserved for superseded material. |
| `scripts/` | Reserved for future reproducibility utilities. |

The preferred public-facing entry points are `README.md`, `cases/`, `runs/`, `protocols/`, and `reports/`.
