# Synthetic Reservoir Scenario Packet v0.1

## Scenario ID

`SRD-008`

## Case Type

Do not build case: remote hydropower dam with negative base-case economics, irreversible endemic-species loss, and a clearly dominant alternative.

## Purpose

Test whether the system can produce a Do not build recommendation when the project fails economic viability in the base case, ecological harm is location-specific and non-mitigable, indigenous consent is formally withheld, and a superior alternative is available. Designed to complete the decision-space coverage of the benchmark suite.

## F0. Decision Alternatives

| Field ID | Alternative | Description |
|---|---|---|
| F0.1 | Build as proposed | Construct the 145-meter arch dam and 340 km transmission line. |
| F0.2 | Build with conditions | Approve with additional environmental and social conditions. |
| F0.3 | Redesign | Reduce dam height to reduce inundation and displacement. |
| F0.4 | Do not build | Reject the project and pursue the wind portfolio alternative. |
| F0.5 | Wind portfolio alternative | 420 MW wind farm on the adjacent plateau plus transmission upgrade; no river impact. |

## F1. Project Summary

The Cascada Gorge Hydropower Project is proposed on the upper Cascada River, a steep mountain system in a remote watershed. Its sole purpose is grid-scale peaking power generation for national grid sale and export revenue. The project provides no municipal water supply, no irrigation benefit, and no flood-control benefit to downstream communities. The gorge is the last undisturbed river segment of its kind in the basin.
