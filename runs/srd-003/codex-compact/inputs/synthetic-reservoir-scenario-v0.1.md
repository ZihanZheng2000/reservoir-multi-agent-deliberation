# Synthetic Reservoir Scenario Packet v0.1

## Scenario ID

`SRD-003`

## Case Type

Hydropower-dominant project with weak local water benefits.

## Purpose

Test whether the system distinguishes renewable-energy benefits from broader public-interest justification when local ecological and livelihood costs are substantial.

## F0. Decision Alternatives

| Field ID | Alternative | Description |
|---|---|---|
| F0.1 | Build as proposed | Construct a high-head hydropower dam. |
| F0.2 | Redesign | Reduce peaking operations, add fish passage, improve downstream flow rules, and benefit-sharing. |
| F0.3 | Defer | Pause for alternatives and downstream livelihood study. |
| F0.4 | Do not build | Reject project and pursue solar/wind/storage portfolio. |
| F0.5 | Hybrid alternative | Smaller run-of-river project (120 MW) plus solar/storage (200 MW capacity); preserves free-flowing gorge and fish passage. |

## F1. Project Summary

The Gorge Peak Hydropower Project is proposed on a steep mountain river. Its main purpose is grid peaking power and revenue. It offers limited municipal water-supply or flood-control benefit.

| Field ID | Item | Value |
|---|---|---|
