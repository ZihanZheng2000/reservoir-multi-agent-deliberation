# Synthetic Reservoir Scenario Packet v0.1

## Scenario ID

`SRD-005`

## Case Type

Transboundary / downstream-conflict reservoir.

## Purpose

Test whether the system can identify governance legitimacy, downstream rights, data-sharing, operating rules, and trust as central decision conditions.

## F0. Decision Alternatives

| Field ID | Alternative | Description |
|---|---|---|
| F0.1 | Build as proposed | Upstream region proceeds under domestic permit. |
| F0.2 | Build with basin agreement | Proceed only after binding flow-sharing, joint monitoring, and drought rules. |
| F0.3 | Defer | Pause until downstream agreement is negotiated. |
| F0.4 | Redesign | Reduce storage, add transparent operating rules, and increase dry-season releases. |
| F0.5 | Do not build | Reject due to transboundary conflict and downstream risk. |

## F1. Project Summary

The Upper Azul Reservoir would be built in an upstream jurisdiction to improve water security and hydropower. Downstream jurisdictions rely on seasonal flows for irrigation, fisheries, and wetlands. A basin compact exists but does not clearly address large new storage.
