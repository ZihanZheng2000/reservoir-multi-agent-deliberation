# Runs

This directory contains model outputs organized by case and run type.

Typical structure:

```text
runs/srd-007/codex-compact/
runs/srd-007/claude-code-compact/
```

SRD-001 has full transcript-style runs:

```text
runs/srd-001/codex-full/
runs/srd-001/claude-code-full/
```

Run manifests may preserve original source paths from the working artifact directory. The canonical GitHub-facing organization is the `runs/` directory itself.
