# Revised Evaluation: SRD-001 Codex Revised Run

## Evaluation Method

The revised evaluation separates:

1. **Shared decision-support quality**: applies to all systems.
2. **Deliberative process quality**: mainly applies to systems that claim to support deliberation.

This avoids claiming MCDA is bad simply because it is not a deliberative system.

**Compact run caveat:** These scores were generated from a compact researcher-synthesized run, not full independently-executed agent transcripts. High subscores (e.g., 8/8 on audit/revision) reflect the structural completeness of the compact summary, not verbatim deliberative transcript quality. Do not treat these scores as equivalent to a full-transcript run until at least one case is expanded.

Independent human expert evaluation is required before treating this as a validated result.

## Interpretation

The revised run still favors the multi-agent system, but the comparison is more defensible because:

- shared quality is reported separately from deliberative quality;
- MCDA receives credit for transparent aggregation and compact decision support;
- single-agent receives credit for concise and calibrated reasoning;
- multi-agent advantage is specifically located in deliberative process quality.

## Remaining Limitations

1. The run is still synthetic.
2. The outputs are still generated by the researcher/assistant, not by independently executed agents.
3. MCDA weights remain provisional.
4. Scores are researcher-coded and need blind human review.
5. Compact summary does not preserve full verbatim agent transcripts.
