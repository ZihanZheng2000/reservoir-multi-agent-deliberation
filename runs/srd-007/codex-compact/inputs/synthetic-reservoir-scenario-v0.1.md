# Synthetic Reservoir Scenario Packet v0.1

## Scenario ID

`SRD-007`

## Case Type

Near-tipping-point case: marginal economics and one incomplete safeguard as the swing condition.

## Purpose

Test whether the system can handle genuine decisional ambiguity — where the project is neither clearly approvable nor clearly rejectable — and produce a specific, time-conditioned recommendation rather than defaulting to a generic Defer. Also tests whether agents can distinguish a near-complete safeguard gap (resolvable in months) from a fundamental project flaw.

## F0. Decision Alternatives

| Field ID | Alternative | Description |
|---|---|---|
| F0.1 | Build | Proceed immediately with construction. |
| F0.2 | Build with conditions | Issue construction approval contingent on completing the livelihood restoration plan within a defined period; other gaps addressed before commissioning. |
| F0.3 | Redesign | Reduce dam height and storage volume to lower cost, reduce displacement, and improve economics. |
| F0.4 | Defer | Pause approval until livelihood plan, downside economic review, and fish passage assessment are complete. |
| F0.5 | Do not build | Reject and rely on improved irrigation efficiency and groundwater development. |

## F1. Project Summary

The Mara Bend Irrigation Reservoir is a medium-scale dam proposed on the Mara River in a semi-arid agricultural region. Its primary purpose is irrigation reliability for smallholder and commercial farmers. Secondary purposes include supplemental urban water supply and small hydropower. The project is at late-feasibility stage with most studies complete.
