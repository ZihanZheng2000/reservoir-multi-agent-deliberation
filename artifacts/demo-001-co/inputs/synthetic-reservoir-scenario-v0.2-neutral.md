## Use Status

- Scenario type: synthetic, idealized, research demo only.
- Real project? no.
- Policy use allowed? no.
- Purpose: test a role-limited multi-agent deliberation workflow for reservoir construction decision support.
- Agent-facing note: this packet contains case facts only. It does not state the expected or preferred decision.

## Scenario ID

`SRD-001-co`

## Project Summary

The proposed **North Fork Multipurpose Reservoir** would be built on the upper North Fork River, 42 km upstream from the regional city of Lantana. The project is proposed as a multipurpose reservoir for urban water supply, flood control, irrigation reliability, and small hydropower generation.

## F0. Decision Alternatives

| Field ID | Alternative | Description |
|---|---|---|
| F0.1 | Build as proposed | Construct the full reservoir and dam as currently designed. |
| F0.2 | Build with conditions | Approve the project only if required studies, safeguards, and operating rules are completed before construction. |
| F0.3 | Redesign | Reduce reservoir height, add stronger environmental-flow guarantees, create fish passage, improve resettlement and compensation, and phase construction after additional studies. |
| F0.4 | Defer | Pause final approval until hydrological, ecological, resettlement, and cost-risk uncertainties are resolved. |
| F0.5 | Do not build | Reject the project and pursue non-dam alternatives. |
| F0.6 | Non-dam portfolio | Demand management, groundwater recharge, watershed restoration, leakage reduction, distributed solar plus battery storage, and smaller off-channel storage. |
