# Case Run Summary: SRD-004

## Case Focus

Flood-control emergency project after repeated disasters.

This case tests whether urgency leads the system to approve irreversible infrastructure before evidence and safeguards are ready.

## Multi-Agent Deliberation Result

### Recommendation

**Phased emergency package now; defer final reservoir approval pending climate, safety, and land-acquisition review.**

### Reasoning Structure

The deliberation recognized that recent floods create a real emergency and that doing nothing is unacceptable. However, engineering, climate, legal, and upstream landowner agents opposed fast-tracking the reservoir itself because the design is conceptual, climate-adjusted flood estimates are incomplete, and emergency acquisition is contested.

The system separated immediate risk-reduction measures from irreversible dam approval.

### Decisive Evidence

| Evidence | Fields | Effect |
|---|---|---|
| Two severe floods affected 86,000 residents. | F2.1, F2.2 | Strongly supports immediate action. |
| Reservoir could reduce 1-in-50-year flood peak by 38%. | F2.3 | Supports reservoir as candidate measure. |
| Extreme flood residual risk remains. | F2.4 | Limits confidence. |
| Concept design and variable foundation data are incomplete. | F3.1, F3.2 | Blocks fast-track build approval. |
| Climate-adjusted flood estimate is incomplete. | F3.3 | Blocks final design confidence. |
| Emergency consultation is shortened and landowner opposition is strong. | F5.3, F5.4 | Creates legitimacy risk. |
| Disaster recovery funding is time-limited. | F6.5 | Supports phased action rather than full deferral. |

### Decisive Uncertainties

| Uncertainty | Fields | Required condition |
|---|---|---|
| Climate-adjusted flood design | F8.1 | Complete before final reservoir approval. |
| Foundation treatment | F8.2 | Complete geotechnical investigation. |
| Emergency acquisition legitimacy | F8.3 | Strengthen consultation and compensation. |
| Downstream operation safety | F8.4 | Complete operating rules and downstream risk test. |

## Single-Agent Baseline Result

### Recommendation

**Proceed with emergency non-structural measures and defer final reservoir approval.**

### Summary

The single-agent baseline also separated urgent flood action from final dam approval. It was strong on immediate practical recommendations, but weaker at preserving competing stakeholder legitimacy concerns.
