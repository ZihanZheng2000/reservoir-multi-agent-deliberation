# Synthetic Reservoir Scenario Packet v0.1

## Scenario ID

`SRD-004`

## Case Type

Flood-control emergency project after repeated disasters.

## Purpose

Test whether the system can handle urgency without abandoning safeguards, alternatives, and long-term risk reasoning.

## F0. Decision Alternatives

| Field ID | Alternative | Description |
|---|---|---|
| F0.1 | Emergency build | Fast-track flood-retention reservoir. |
| F0.2 | Phased emergency package | Immediate levee repair, warning systems, zoning, wetland restoration, and final reservoir decision after studies. |
| F0.3 | Redesign | Smaller retention basin plus floodplain restoration. |
| F0.4 | Defer | Complete full studies before any capital works. |
| F0.5 | Do not build | Use non-structural flood measures only. |

## F1. Project Summary

The Lower Vale Flood Retention Reservoir is proposed after two severe floods in five years. Public pressure for immediate action is high. The project would temporarily store floodwaters upstream of several towns.
