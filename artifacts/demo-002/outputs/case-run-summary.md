# Case Run Summary: SRD-002

## Case Focus

Low-conflict off-channel storage / existing reservoir expansion.

This case tests whether the system can recommend approval when social and ecological risks are comparatively low.

## Multi-Agent Deliberation Result

### Recommendation

**Build with conditions through phased approval.**

### Reasoning Structure

The expert and stakeholder agents largely agreed that the project is materially different from a high-conflict mainstem dam. It expands existing off-channel storage, does not block the main river, affects no known critical habitat, has a completed EIA, and has limited displacement.

The main conflicts were not whether the project should exist, but how approval should be conditioned:

- water utility and government favored approval because additional firm yield exceeds the projected deficit;
- environmental NGO gave qualified support if the wetland buffer is enforceable;
- affected grazing users accepted compensation if access routes remain available;
- local government conditioned support on low-income tariff protection;
- climate and engineering experts required two-year drought refill testing; seismic reinforcement of the existing embankment was classified as a standard retrofit condition, not a design blocker;
- finance office supported the project if the climate adaptation grant is secured.

### Decisive Evidence

| Evidence | Fields | Effect |
|---|---|---|
| Additional firm yield exceeds projected deficit. | F2.3, F2.4 | Supports approval. |
| Off-channel design does not block main river. | F3.1 | Reduces ecological conflict. |
| No known endemic species or critical habitat affected. | F3.3 | Reduces ecological blocker risk. |
| Physical displacement is limited to 42 households. | F4.1 | Lower social conflict than SRD-001. |
| Final EIA is completed with conditional approval recommendation. | F6.3 | Supports governance readiness. |
| Base and downside BCR remain above 1.0. | F5.3 | Supports economic viability. |

### Decisive Uncertainties

| Uncertainty | Fields | Required condition |
|---|---|---|
| Two-year drought refill performance | F8.1 | Stress-test dynamic refill rule. |
| Tariff protection | F5.5, F8.2 | Create low-income subsidy or tariff cap. |
| Wetland buffer enforcement | F3.4, F8.3 | Make buffer legally enforceable. |
| Seismic reinforcement cost | F6.2, F8.4 | Confirm retrofit cost within preliminary USD 8–11 million estimate and include in project budget. |

### Multi-Agent Final Position

The system does **not** default to Defer/Reject. It recommends approval, but approval is phased and conditional.

## Single-Agent Baseline Result

### Recommendation

**Approve with conditions.**

### Summary

The single-agent baseline identified the project as a comparatively low-conflict storage expansion with strong water-security justification. It recommended approval with conditions for climate stress testing, tariff protection, wetland buffer enforcement, and seismic reinforcement.

### Limitation

The single-agent result was efficient and mostly aligned with multi-agent output, but it did not preserve separate stakeholder positions, especially grazing users, low-income tariff concerns, and environmental conditional support.

## Case Interpretation

SRD-002 is important because it shows the multi-agent system is not simply biased toward rejection or deferral. When the scenario has low ecological and social blockers, better governance readiness, and robust economics, the system can recommend conditional approval.
