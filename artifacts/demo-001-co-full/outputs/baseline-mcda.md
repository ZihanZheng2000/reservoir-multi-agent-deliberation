# MCDA Baseline: SRD-001-co-full

## Scoring Method

The MCDA baseline assigns 1-5 scores to decision criteria, then maps the weighted average to a recommendation. Scores are researcher-coded from the neutral scenario packet.

## Criterion Scores

| Criterion | Score 1-5 | Rationale |
|---|---:|---|
| water_supply_benefit | 3 | Base yield exceeds demand gap, but severe drought weakens reliability. |
| flood_control_benefit | 3 | Useful 1-in-20 reduction; limited for extremes. |
| energy_benefit | 3 | 82 MW and 265 GWh useful but dry-year generation falls. |
| economic_viability | 2 | BCR 1.18 base, 0.86 downside; P90 cost high. |
| engineering_feasibility | 2 | Feasible concept but abutment, EAP, spillway, access issues unresolved. |
| ecological_impact_acceptability | 1 | Endemic fish, habitat loss, fish passage uncertainty. |
| social_impact_acceptability | 1 | Major displacement, sacred sites, incomplete livelihood plan. |
| legal_governance_readiness | 2 | Policy fit but EIA/RAP/biodiversity/consultation incomplete. |
| climate_robustness | 2 | Climate-adjusted inflow and spillway analysis incomplete. |
| stakeholder_support | 2 | Urban/policy support but affected residents and NGO oppose. |
| evidence_confidence | 3 | Many facts available, but key approval studies incomplete. |

Weighted score: **2.34 / 5.00**

## Recommendation Mapping

| Score band | Recommendation |
|---|---|
| >= 4.0 | Build |
| 3.2-3.9 | Build with conditions |
| 2.4-3.1 | Redesign / Defer |
| < 2.4 | Do not build as proposed |

The score is near the boundary between **Do not build as proposed** and **Redesign / Defer**. Because public benefits are credible, the MCDA recommendation is:

**Do not build as proposed; major redesign and approval deferral required.**

## Limitation

MCDA correctly flags that the current package is weak, but it cannot naturally express which issues are construction-blocking, which are redesign conditions, and which are commissioning conditions. It also hides the difference between ecological, social, and financing veto points inside one weighted average.
