# Synthetic Reservoir Scenario Packet v0.1

## Scenario ID

`SRD-006`

## Case Type

Clean approval case: small off-channel water security reservoir.

## Purpose

Test whether the system can produce a Build or Build with minor operating conditions recommendation when no structural blockers remain, safeguards are complete, and economics are robust across all scenarios. Designed to expose over-conservatism if the system adds unnecessary deferral.

## F0. Decision Alternatives

| Field ID | Alternative | Description |
|---|---|---|
| F0.1 | Build | Construct as designed and commissioned under agreed operating rules. |
| F0.2 | Build with operating conditions | Approve construction; attach minor operating rule and monitoring conditions to be resolved before first operation, not before construction. |
| F0.3 | Redesign minor components | Reduce spillway capacity or adjust intake elevation to lower cost. |
| F0.4 | Defer | Pause construction approval until long-term sediment management plan is formally adopted. |
| F0.5 | Do not build | Reject and rely on demand management and managed aquifer recharge. |

## F1. Project Summary

The East Ridge Water Security Reservoir is a small off-channel storage facility fed by a seasonal diversion canal from Linden Creek. It would close a confirmed dry-season water deficit for the town of Ravello and surrounding smallholder farms in a semi-arid upland region. The facility does not block the main river.
